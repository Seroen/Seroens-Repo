Recommended Level Editors

Henry's Leditor

Henry's level editor encourages a fast workflow. Its fairly similar to Rained, but it's a lot more customizable and the creator is always open for suggestions.
DM human_resource on Discord if you want to provide feedback.

Rained

Rained is my favorite level editor because it encourages a fast work flow and the creator is always open to suggestions.
DM chromosoze on discord if you want to provide feedback.

Ogscule Editor

The Ogscule level editor is a smoother and better version of the Rain World Editor Plus. The creator is always open to suggestions.
DM mayhemmmwiththreems on Discord to provide feedback.

Other Level Editors

Community Editor

This is the Community editor, formerly know as the Drought editor. It's very similar to the Official and Downpour editors. It's most unique feature is the ability to add custom effects easier than with other editors.

RWE+

This is first custom level editor, made by Timofey. He is currently working on a better editor, so I would strongly recommend not using this one

Official Editor

This is the original level editor released in 2018. It's here for achivial purposes, and its use is discouraged.

Downpour Editor

This is the downpour level editor released in 2023. It's here for achivial purposes, and its use is discouraged.

Discord Servers

Main Server

This is the official Rain World server. It has close ties to the community and you can also get started on your modding journey.

Modding Server

This is the official modding server for Rain World, where people will help in your journey to be a great level creator.

Useful Resources

Modding Wiki

The Rain World modding wiki, a place to share and obtain information about modding Rain World. Feel free to contribute!

Main Wiki

The official Rain World wiki, containing information about all items in the game.

Mods Wiki

The Rain World mods wiki, a place to collect information on content mods for Rain World. Feel free to contribute!

Region Lease

A spreadsheet used to help ensure region mods don't conflict with each other.

Palette Guide

A guide on how palettes work and how to make them.

Tiles & Props Creation Guide

A guide on how to make custom tiles and props.

Tile Init Editor

A tool that gives a simple and easy-to-use UI for creating Level Editor tile init lines.

Prop Renderer

A scene and rendering workflow for Blender that slices and converts closed meshes into prop sprite sheets that can be used with level editors.

Drizzle

Faster renderer that comes with, and is used by most level editors. Not used by Official, Downpour, and Community editors.

Level Design Tutorial

While this tutorial is outdated it still contains useful information.

Region Setup Tutorial

While this tutorial is outdated it still contains useful information.

World Editor

A tool used to quickly connect rooms, add creature spawns and create in-game maps for regions, though it its still recommended to learn how to do so without it.

World Editor Tutorial

An outdated but hopefully still useful guide for world editor.

Cornifer

A tool used to make wiki style maps.

Base Game Interactive Map

Website with detailed maps of all base game and MSC regions.

Mod Interactive Map

Website with detailed maps of several modded regions.

Room Prompt Generator

This can generate random room prompts if you are looking for inspiration.

Render Editor

A tools that allows for limited editing of the pre-existing Rain World levels, such as editing pipe interconnections or changing water levels.

Adobe Director

The software used to make the Official, Downpour, and Comms Editors. It can be used to edit the Comms code to add new effects. Due to it being abandoned by Adobe, the trial must be reset manually every 30 days.
